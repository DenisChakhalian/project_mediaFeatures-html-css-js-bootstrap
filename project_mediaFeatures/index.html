Описание

Media features используются тогда, когда нужно создать различные CSS-стили, для разных устройств по типу отображения (например, для принтера, монитора или смартфона), а также конкретных характеристик устройства (например, ширины окна просмотра браузера), или внешней среды (например, внешнее освещение). Учитывая огромное количество подключаемых к интернету устройств, медиа-запросы являются очень важным элементом при создании веб-сайтов и приложений, которые будут правильно работать на всех доступных устройствах, которые есть у Ваших пользователей.

Медиа-запросы - это характеристика определенного пользователя, устройства вывода или окружения. Например, Вы можете применить выбранные стили только для широкоэкранных мониторов, компьютеров с мышью, или для устройств, которые используются в условиях слабой освещенности.

Благодаря медиа-запросам можно создавать адаптивный макет. Макет регулируется в зависимости от разрешение экрана и окна браузера, изменяя при этом ширину макета, разрешение изображений, количество колонок и текста. Медиа-запросы устанавливают ширину макета и при достижении заданного значения (например, при уменьшении окна браузера или при просмотре на девайсе с указанным размером) будет применяется совершенно иной стиль.

Синтаксически медиа-запросы напоминают свойства CSS: они имеют имена и принимают определенные значения. Однако есть несколько важных различий между свойствами и функциями мультимедиа:

  • Свойства используются в объявлениях для предоставления информации о том, как представить документ. Медиа-запросы используются в выражениях для описания требований устройства вывода.
  • Большинство медиа-запросов принимают необязательные префиксы «min» или «max» для выражения ограничений «больше или равно» и «меньше или равно». Данный синтаксис используется для исключения символов «<» и «>», которые могут быть не совместимы с HTML и XML. Те медиа-запросы, которые принимают префиксы, чаще всего будут использоваться с префиксами, но также могут использоваться отдельно...
