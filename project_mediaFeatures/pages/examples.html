Пример 1. Медиа-запросы для меню

В этом примере мы используем медиа-запросы для создания адаптивного меню навигации, которое меняется в дизайне на различных размерах экрана:

            
    <!DOCTYPE html>
    <html lang="ru">
        <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1">
            <title>Пример 1</title>
            <style>
                * {
                    box-sizing: border-box;
                }
        
                /* Стиль верхней панели навигации */
                .topnav {
                    overflow: hidden;
                    background-color: #333;
                }
        
                /* Стиль верхних навигационных ссылок */
                .topnav a {
                    float: left;
                    display: block;
                    color: #f2f2f2;
                    text-align: center;
                    padding: 14px 16px;
                    text-decoration: none;
                }
        
                /* Изменение цвета при наведении */
                .topnav a:hover {
                    background-color: #ddd;
                    color: black;
                }
        
                /* На экранах, которые 600px шириной или меньше, сделать ссылки меню стек поверх друг друга, а не рядом друг с другом */
                @media screen and (max-width: 600px) {
                    .topnav a {
                        float: none;
                        width: 100%;
                    }
                }
            </style>
        </head>
        <body>
            <div class="topnav">
                <a href="#">Главная</a>
                <a href="#">Не главная</a>
                <a href="#">Третья</a>
            </div>
            <h1>С другой стороны начало повседневной работы</h1>
            <p>по формированию позиции обеспечивает широкому кругу специалистов участие в формировании дальнейших направлений развития проекта.
            Практический опыт показывает, что постоянный количественный рост и сфера нашей активности способствует повышению актуальности форм
            воздействия? Не следует, однако, забывать о том, что реализация намеченного плана развития позволяет оценить значение всесторонне
            сбалансированных нововведений. Задача организации, в особенности же повышение уровня гражданского сознания влечет за собой процесс
            внедрения и модернизации дальнейших направлений развитая системы массового участия. Практический опыт показывает, что реализация
            намеченного плана развития представляет собой интересный эксперимент проверки системы масштабного изменения ряда параметров.</p>
        </body>
    </html>

Пример 2. Медиа-запросы для столбцов

В этом примере мы создаем макет, который варьируется от четырех, в два столбца и столбцы полной ширины, в зависимости от различных размеров экрана:

            
    <!DOCTYPE html>
    <html>
        <head>
            <meta name="viewport" content="width=device-width, initial-scale=1">
            <title>Пример 2</title>
            <style>
            * {
                box-sizing: border-box;
            }
        
            /* Создайте четыре одинаковых столбца, которые плавают рядом друг с другом */
            .column {
                float: left;
                width: 25%;
                padding: 20px;
            }
        
            /* Очистить поплавки после колонн */
            .row:after {
                content: "";
                display: table;
            }
        
            /* На экранах шириной 992px или меньше перейдите от четырех столбцов к двум столбцам */
            @media screen and (max-width: 992px) {
                .column {
                width: 50%;
                }
            }
        
            /* На экранах, которые имеют ширину 600 пикселей или меньше, сделайте стек столбцов поверх друг друга, а не рядом друг с другом */
            @media screen and (max-width: 600px) {
                .column {
                width: 100%;
                }
            }
            </style>
        </head>

            nоn datur. </p>

                <h2>Колонка 1</h2>
            </div>

Пример 3. Браузер и печать

В данном примере вводится два стиля — один для изменения вида элементов при их обычном отображении в браузере, а второй — при выводе страницы на печать. При этом облик документа для разных носителей может сильно различаться между собой:

            
    <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" 
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
    <html xmlns="http://www.w3.org/1999/xhtml">
        <head>
            <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
            <title>Пример 3</title>
            <style type="text/css">
                @media screen { /* Стиль для отображения в браузере */
                    body {
                        font-family: Arial, Verdana, sans-serif; /* Рубленый шрифт */
                        font-size: 0.9em; /* Размер шрифта */
                        color: #804900; /* Цвет текста */
                    }
                    h1 {
                        background: #faf0e6; /* Цвет фона под текстом */
                        border: 2px dashed #ff9a9a; /* Рамка вокруг заголовка */
                        color: #2a9939; /* Цвет текста */
                        padding: 7px; /* Поля вокруг текста */
                    }
                    h2 {
                        color: #3b2a86; /* Цвет текста */
                        margin: 0; /* Убираем отступы */
                    }
                    p {
                        margin-top: 0.5em; /* Отступ сверху */
                    }
                }
                @media print { /* Стиль для печати */
                    body {
                        font-family: Times, 'Times New Roman', serif; /* Шрифт с засечками */
                    }
                    h1, h2, p {
                        color: #000; /* Черный цвет текста */
                    }
                }
            </style>
        </head>
        <body>
            <h1>Апостериори</h1>
            <h2>гравитационный парадокс</h2>
            <p> Интеллект естественно понимает под собой интеллигибельный закон внешнего мира, открывая новые горизонты. Дискретность амбивалентно
            транспонирует гравитационный парадокс. Отсюда естественно следует, что автоматизация дискредитирует предмет деятельности. Согласно
            мнению известных философов, дедуктивный метод естественно порождает и обеспечивает мир, tertium nоn datur.</p>
            <p>Созерцание непредсказуемо. Аксиома силлогизма, по определению, представляет собой неоднозначный предмет деятельности.
            Гедонизм осмысляет дедуктивный метод. Надстройка нетривиальна. Дедуктивный метод решительно представляет собой бабувизм. Структурализм
            абстрактен</p>
        </body>
    </html>

Пример 4. Использование max-width

При уменьшении окна до 600 пикселов и меньше меняется цвет фона веб-страницы:

            
    <!DOCTYPE html>
    <html>
        <head>
            <meta charset="utf-8">
            <title>Пример 4</title>
            <style>
                body { background: #ffe693; }
                @media screen and (max-width: 600px) {
                    body { background: rgb(255, 0, 76); }
                }
            </style>
        </head>
        <body>
            <p>Структурализм абстрактен. Надстройка нетривиальна. Согласно мнению известных философов, дедуктивный метод естественно порождает и
            обеспечивает мир, tertium nоn datur. Надстройка нетривиальна. Апостериори, гравитационный парадокс амбивалентно понимает под собой
            интеллигибельный знак. Интеллект естественно понимает под собой интеллигибельный закон внешнего мира, открывая новые горизонты.  </p>
        </body>
    </html>

            <title>Пример 5</title>
            <style>
                div.example {
                    background-color: rgb(119, 255, 187);
                    padding: 20px;
                }

            <p>Гравитационный парадокс амбивалентно понимает под собой интеллигибельный знак. Дедуктивный метод решительно представляет собой
            бабувизм. Гедонизм осмысляет дедуктивный метод. Интеллект естественно понимает под собой интеллигибельный закон внешнего мира, открывая
            новые горизонты.</p>

Пример 6. Изменение размера шрифта

Если размер экрана более 600 пикселей в ширину, устанавливается размер шрифта <div> в 80 пикселей, а если размер экрана 600px широкий, или меньше, установите размер шрифта <div> в 30px:

            
    <!DOCTYPE html>
    <html>
        <head>
            <meta name="viewport" content="width=device-width, initial-scale=1">
            <title>Пример 6</title>
            <style>
                div.example {
                    background-color: rgb(255, 203, 203);
                    padding: 20px;
                }

                @media screen and (min-width: 600px) {
                    div.example {
                    font-size: 80px;
                    }
                }

                @media screen and (max-width: 600px) {
                    div.example {
                    font-size: 30px;
                    }
                }
            </style>
        </head>
        <body>
            <div class="example">Аксиома силлогизма.</div>
            <p>Дедуктивный метод решительно представляет собой бабувизм. Смысл жизни, следовательно, творит данный закон внешнего мира. Аксиома
            силлогизма, по определению, представляет собой неоднозначный предмет деятельности. </p>
        </body>
    </html>
