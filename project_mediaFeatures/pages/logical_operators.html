Логические операторы

Каждая медиа-функция проверяет одну особенность браузера или устройства.
Логические операторы и их описание
Имя Описание Примечание
width Ширина области просмотра.
height Высота окна просмотра.
aspect-ratio Соотношение ширины к высоте в области просмотра.
orientation Ориентация области просмотра.
resolution Плотность пикселей устройства вывода.
scan Процесс сканирования выходного устройства.
grid Является ли устройство сеткой или растровым изображением?
update-frequency Как быстро (если это вообще возможно) устройство вывода может изменить внешний вид контента. Добавлено в Media Queries Level 4.
overflow-block Как устройство вывода обрабатывает контент, который выходит за пределы области просмотра вдоль оси блока? Добавлено в Media Queries Level 4.
overflow-inline Можно ли прокрутить содержимое, которое выходит за пределы области просмотра вдоль встроенной оси? Добавлено в Media Queries Level 4.
color Количество бит на компонент цвета устройства вывода или ноль, если устройство не является цветом.
color-index Количество записей в таблице соответствия цветов устройства вывода или ноль, если устройство не использует такую ​​таблицу.
display-mode Режим отображения приложения, указанный в элементе отображения manifest веб-приложения. Определяется в спецификации manifest веб-приложения
monochrome Биты на пиксель в монохромном кадровом буфере устройства вывода или 0, если устройство не монохромное.
inverted-colors Пользователь или базовая ОС инвертируют цвета? Добавлено в Media Queries Level 4.
pointer Является ли основной механизм ввода указательным устройством, и если да, то насколько он точен? Добавлено в Media Queries Level 4.
hover Позволяет ли основной механизм ввода пользователю наводить курсор на элементы? Добавлено в Media Queries Level 4.
any-pointer Является ли какой-либо доступный механизм ввода указательным устройством, и если да, то насколько он точен? Добавлено в Media Queries Level 4.
any-hover Любой доступный механизм ввода позволяет пользователю наводить курсор на элементы? Добавлено в Media Queries Level 4.
light-level Текущий уровень освещенности. Добавлено в Media Queries Level 4.
scripting Доступны ли скрипты (например, JavaScript)? Добавлено в Media Queries Level 4.
device-width Ширина поверхности рендеринга устройства вывода. Устаревшее в Media Queries Level 4.
device-height Высота рендеринга поверхности устройства вывода. Устаревшее в Media Queries Level 4.
device-aspect-ratio Соотношение ширины к высоте устройства вывода. Устаревшее в Media Queries Level 4.
-webkit-device-pixel-ratio Количество пикселей физического устройства на пиксель CSS. Нестандартное; Если возможно, используйте вместо этого функцию transform.
-webkit-transform-3d Поддерживает ли CSS 3D transform? Нестандартное; WebKit/Blink-specific.
-webkit-transform-2d Поддерживает ли CSS 2D transform? Нестандартное; WebKit-specific.
-webkit-transition Поддерживает ли CSS transition? Нестандартное; WebKit-specific.
-webkit-animation Поддерживает ли CSS animation? Нестандартное; WebKit-specific.
